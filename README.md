# Networked Chat Application

![Application Demo](images/chat-demo.png)

*A real-time distributed chat application showing multiple clients connected and messaging*

## Live Demo
The screenshot above shows:
- Multiple clients connected simultaneously
- Real-time message broadcasting
- User interface in action
- System notifications for client connections
- Message history displayed

## Features
- Distributed Architecture - Multiple client support
- Real-time Messaging - Instant message delivery
- Connection Management - Tracks client connections/disconnections
- JavaFX GUI - Modern user interface
- Network Communication - Socket-based client-server model

## Technologies
- Java & JavaFX
- Socket Programming
- Multi-threading
- Maven Build System
- Client-Server Architecture
